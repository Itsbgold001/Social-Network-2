# Social-Network
A social network based on php OOP and mySQL.
<br />
# How to use.
1) Create a database for the project. <br />
2) Edit this file <code>core/init.php</code> and add your database information. <br />
3) Execute <code>config.sql</code> file. <br />
4) have fun!
